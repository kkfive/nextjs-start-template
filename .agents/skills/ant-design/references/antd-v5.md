# Ant Design v5 参考

## 适用范围

- 适用于 `antd@^5` 的存量项目与维护场景。
- React 版本以项目现状为准，升级到 v6 前需确认兼容性与迁移成本。

## 关键差异（相对 v6）

- v6 引入更多语义化 DOM 与 `classNames` / `styles` 定制能力，v5 在部分组件上可能不完整。
- v6 迁移需要参考 `https://ant.design/docs/react/migration-v6` 的官方清单。

## 使用建议

- 新功能优先参考 v6 文档与实现，v5 仅用于维护与回滚。
- 避免引入新的全局样式覆盖，尽量使用既有主题与 token 体系。

## 参考文档

- `https://ant.design/docs/react/migration-v6`
- `https://ant.design/docs/react/introduce`
