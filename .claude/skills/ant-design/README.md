# Ant Design Skill

Ant Design 6.x 组件选择、主题配置、CSS-in-JS/SSR 设置指南。

## 使用场景

- 构建或审查 Ant Design UI
- 配置 ConfigProvider 和主题
- 解决 Ant Design 相关问题
- 选择合适的组件实现功能

## 调用方式

```bash
/ant-design
```

## 覆盖范围

- 组件选择和使用
- 主题配置（ConfigProvider）
- CSS-in-JS 和 SSR 设置
- 常见组件高级用法（Form、Table、Select、Tree、Upload）
- Ant Design 5.x 到 6.x 迁移
