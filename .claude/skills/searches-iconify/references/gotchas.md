# Gotchas

## 关键词

- **只查原词** → 0 结果是必然；至少加 2 个同义词
- **中文直查** → Iconify API 不识别中文；先翻译
- **关键词太相似** → `setting` 与 `settings` 结果几乎一样；去重

## 并行

- **分多次响应执行** → 慢、用户体验差；必须同一响应并发
- **拿到部分结果就输出** → 排序失真；等全部完成
- **中间穿插其他工具调用** → 打断并行；先发完所有 curl 再做别的

## 聚合

- **不去重** → 同一图标在多组重复出现，推荐失真
- **不标记 0 结果** → 用户不知道是没搜到还是漏写；明示 "(找到 0 个)"
- **推荐里堆 20 个** → 信息过载；保留 5 个

## 输出格式

- **加导语 / 结尾客套话** → 违反模板
- **预览 URL 写错** → `api.iconify.design/{collection}:{name}.svg`，冒号分隔，不是斜杠
- **图标集大小写错** → 集名 lower-case，如 `mdi`、`lucide`、`tabler`、`carbon`

## API

- **`limit=20` 不够** → 必要时翻到 50；但通常 20 已饱和
- **API 偶尔超时** → 单个 curl 失败时其他仍输出，标记该关键词失败
- **rate limit** → Iconify 公网 API 无明显限流；如遇 429 减少关键词或加 sleep
